Projects

PROXYAUTH

A SECURITY CAPSTONE PROJECT

A Continuous Authentication System that uses Bluetooth on your Smartphone Device to provide authentication to successfully login on the user's computer. ProxyAuth is equipped with a Bluetooth Continuous Authentication Scheme. The users aren't required to log off the computer, the computer will lock if it cannot detect the device within the Bluetooth range.

Langauge & Tools: Kotlin, C, PAM (Pluggable Authentication Module), Bluetooth Programming, D-Bus

WAREHOUSE WARS

WEB-BASED ARCADE GAME

A player (cool face) is able to move into any empty, adjacent square. If the empty space has a box in it, it could be moved by player. The player can move a long line of boxes (used object recursion for that). Monster can kill the player going to the same square. The goal of the game is to completely sorround each monster by given boxes. The player has to authenticate itself to play and to store the scores in a scoreboard. A good example of Full Stack Development from front end to back end.

Langauge & Tools:

  • Front end: HTML5, CSS3, JavaScript and later replaced with React
  • Back end: Ajax (JSON transport), JQuery, JavaScript, Node+Express
  • Database: SQLite3 and later with MongoDB
